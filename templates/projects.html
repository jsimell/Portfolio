Traffic Simulator

A group project in the Aalto University Object Oriented Programming with C++ course. On top of pure C++ programming skills, I learned a lot about working as part of a developer team. Furthermore, I learned what is required to construct a C++ program from scratch.


My main responsibilities in the project were: creating initial implementations for the TrafficLight and Intersection classes, implementing the functionality of adding intersections and traffic lights to the city, creating the example city models, handling of situations where the input file is invalid, configuration management, setting up the simple testing environment, and debugging.
